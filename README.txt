## Creacion de un sitio web para plataformas online de contenido digital.

Consideraciones generales: 

- No se permite el uso de librerias y framwork externos 
- Se debe crear una o mas hojas de estilo CSS
- La pagina web debe visualizare correctamente en las versiones actuales de Google Chrome, Mozilla Firefox y Microsoft Edge.
- No se requiere el uso de interacion mediante JavaScript
